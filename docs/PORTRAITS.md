# PORTRAITS.md
> Читать при работе с `portraitSvg` в `js/config.js`

Каждый персонаж имеет SVG-портрет 100×100 — бюст в стиле fantasy token.
Хранится как строка в `CONFIG.characters[n].portraitSvg`.
Отображается в токене (круглая обрезка 64×64) и в карточке (80×80).

---

## Общий шаблон SVG-портрета

```svg
<svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  <!-- 1. Фон — тёмный градиент с цветовым акцентом класса -->
  <defs>
    <radialGradient id="bg_ID" cx="50%" cy="70%" r="80%">
      <stop offset="0%" stop-color="{accentColor}" stop-opacity="0.3"/>
      <stop offset="100%" stop-color="#06060a"/>
    </radialGradient>
  </defs>
  <rect width="100" height="100" fill="url(#bg_ID)"/>

  <!-- 2. Тело / одежда персонажа -->
  <!-- 3. Голова и лицо -->
  <!-- 4. Характерный атрибут класса (оружие, шляпа, символ) -->

  <!-- 5. Виньетирование по краям -->
  <defs>
    <radialGradient id="vignette_ID" cx="50%" cy="50%" r="50%">
      <stop offset="60%" stop-opacity="0"/>
      <stop offset="100%" stop-color="#000" stop-opacity="0.7"/>
    </radialGradient>
  </defs>
  <circle cx="50" cy="50" r="50" fill="url(#vignette_ID)"/>
</svg>
```

## Описание портретов героев

### Паладин — accentColor: #c8a000
- **Фон:** тёплый золотистый градиент снизу
- **Тело:** широкие золотые пластинчатые наплечники, белый плащ
- **Голова:** светлые волосы, квадратное лицо, голубые глаза
- **Атрибут:** рукоять меча входит в кадр снизу-слева, белое свечение вокруг
- **Эффект:** лёгкий белый halo вокруг головы (divine aura)

### Бард — accentColor: #9b59b6
- **Фон:** фиолетовый градиент
- **Тело:** камзол с кружевным воротником, бордовый жилет
- **Голова:** треугольная фиолетовая шляпа с пером, рыжие кудри, карие глаза
- **Атрибут:** гриф лютни выглядывает из-за плеча
- **Эффект:** маленькие музыкальные ноты ♪ вокруг шляпы

### Клирик — accentColor: #3498db
- **Фон:** синий градиент
- **Тело:** синее балахон священника с капюшоном, откинутым назад
- **Голова:** седая борода, морщины, мудрые глаза, лысина
- **Атрибут:** посох с светящимся синим кристаллом входит снизу
- **Эффект:** тихое синее свечение вокруг кристалла

### Рейнджер — accentColor: #27ae60
- **Фон:** тёмно-зелёный градиент
- **Тело:** зелёный кожаный плащ с капюшоном (надет на голову)
- **Голова:** острые эльфийские уши, зелёные глаза, тёмные волосы за ушами
- **Атрибут:** стрела за плечом, кончик лука виден сбоку
- **Эффект:** листья и еловые ветки на фоне

### Плут — accentColor: #e74c3c
- **Фон:** почти чёрный с красным отблеском
- **Тело:** чёрный кожаный доспех, тёмный капюшон надвинут на лоб
- **Голова:** смуглая кожа, шрам через левую бровь, прищуренный острый взгляд
- **Атрибут:** кинжал виден у пояса, рукоять торчит снизу кадра
- **Эффект:** тени на лице акцентированы

## Описание портретов боссов

### Вожак бандитов — accentColor: #5a3e2b
- **Фон:** коричнево-серый
- **Тело:** потёртая кожаная накидка, ржавые заклёпки
- **Голова:** грубое лицо, густая щетина, несколько шрамов, злобный прищур
- **Атрибут:** топор на плече, виден краем
- **Эффект:** грязь на лице, кровь на накидке

### Тёмный рыцарь — accentColor: #2c3e50
- **Фон:** тёмно-серый металлик
- **Тело:** вороные зазубренные доспехи, шипы на наплечниках
- **Голова:** закрытый чёрный шлем, через узкую прорезь видно красное свечение глаз
- **Атрибут:** намёк на огромный двуручный меч сзади
- **Эффект:** красное свечение из-под забрала

### Некромант — accentColor: #4a235a
- **Фон:** тёмно-фиолетовый с еле видными звёздами
- **Тело:** тёмная мантия с рунами, капюшон со звёздами
- **Голова:** жёлтая сухая кожа, пустые белые светящиеся глаза (нет зрачков)
- **Атрибут:** символ черепа с рунами на груди, кости-пальцы
- **Эффект:** фиолетовый туман вокруг рук

### Тёмный Лорд — accentColor: #7b0000
- **Фон:** абсолютно чёрный с красными прожилками, как трещины в камне
- **Тело:** массивные чёрные доспехи с красными рунами, светятся
- **Голова:** корона из костей, красные горящие глаза, нет лица — только тьма
- **Атрибут:** красное свечение всего силуэта
- **Эффект:** красные руны пульсируют (можно через CSS animation на SVG-элемент)

## SVG-иконки кубиков (dice.js)

Маленькие геометрические SVG 24×24px для форм ввода.
Цвет: `#c8a000` (золото), обводка `#8b6914`, фон прозрачный.

| Кубик | SVG-форма                        | Число внутри |
|-------|----------------------------------|--------------|
| d4    | Равносторонний треугольник       | «4»          |
| d6    | Квадрат с закруглёнными углами   | «6»          |
| d8    | Ромб (квадрат 45°)               | «8»          |
| d10   | Вытянутый ромб                   | «10»         |
| d12   | Пятиугольник                     | «12»         |
| d20   | Вытянутый шестиугольник (вертикальный) | «20»    |

Число рисовать по центру фигуры: `font-family: 'VT323', monospace`, size 9–10px, fill: `#fff`.
